app.js
 ├── loads environment variables
 ├── connects middleware
 ├── connects routes
 ├── connects database
 └── starts server


 User submits form
        ↓
app.js receives request
        ↓
productRoutes.js matches POST route
        ↓
uploadPdf middleware reads PDF
        ↓
productController.createProduct runs logic
        ↓
Supabase stores PDF
        ↓
Product model saves data in PostgreSQL
        ↓
User redirected to /home


pseudocode

A. app.js
START app
LOAD environment variables
CREATE Express app
ADD middleware
REGISTER product routes
CONNECT to database
START server on PORT


B. routes/productRoutes.js
WHEN user sends POST request to /products
RUN uploadPdf middleware
THEN run createProduct controller


C. middlewares/uploadPdf.js
RECEIVE uploaded file
CHECK file type is PDF
STORE file temporarily in memory
PASS file to controller as req.file


D. controllers/productController.js
GET current user from session
IF user does not exist
    redirect to home with error

GET product name and description
IF fields are empty
    redirect to home with error

IF PDF exists
    create unique file name
    upload PDF buffer to Supabase
    get public PDF URL

CREATE product row in database
SAVE product name, description, user ID, and PDF URL
redirect to home with success



E. models/Product.js
DEFINE Product table
FIELDS:
    id
    product_name
    product_description
    user_id
    pdf_path
    createdAt
    updatedAt
